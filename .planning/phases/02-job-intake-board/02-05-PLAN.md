The toolbar renders when `selectedJobIds.size > 0`. It contains:
- Text: "X jobs selected"
- A `<select>` dropdown listing all stages by name
- "Move" button that calls `bulkMoveJobs` with the selected IDs and chosen stage ID
- "Clear" button that clears the selection

Implementation notes:
- Import `bulkMoveJobs` from '@/actions/jobs'
- Add `const [bulkTargetStageId, setBulkTargetStageId] = useState('')`
- After bulk move succeeds: clear selectedJobIds (set to empty Set), clear bulkTargetStageId
- After bulk move: call `revalidatePath` is handled by the server action — the optimistic board state will refresh on next navigation or can be manually reset by clearing selectedJobIds and triggering a re-fetch; for simplicity, after bulkMoveJobs resolves, reset selectedJobIds to empty Set (the board will show stale data until next navigation, which is acceptable for v1)
- The toolbar div: `className="px-4 pt-2 flex items-center gap-3"` above the columns div

Example toolbar structure:
```tsx
{selectedJobIds.size > 0 && (
  <div className="px-4 py-2 border-b flex items-center gap-3 bg-muted/30">
    <span className="text-sm text-muted-foreground">
      {selectedJobIds.size} job{selectedJobIds.size !== 1 ? 's' : ''} selected
    </span>
    <select
      value={bulkTargetStageId}
      onChange={(e) => setBulkTargetStageId(e.target.value)}
      className="text-sm border rounded px-2 py-1 bg-background"
    >
      <option value="">Move to stage...</option>
      {stages.map((s) => (
        <option key={s.id} value={s.id}>{s.name}</option>
      ))}
    </select>
    <button
      type="button"
      onClick={() => {
        if (!bulkTargetStageId) return
        startTransition(async () => {
          await bulkMoveJobs([...selectedJobIds], bulkTargetStageId)
          setSelectedJobIds(new Set())
          setBulkTargetStageId('')
        })
      }}
      disabled={!bulkTargetStageId}
      className="text-sm px-3 py-1 rounded bg-[var(--brand)] text-white hover:opacity-90 disabled:opacity-40 transition-opacity"
    >
      Move
    </button>
    <button
      type="button"
      onClick={() => setSelectedJobIds(new Set())}
      className="text-sm text-muted-foreground hover:text-foreground"
    >
      Clear
    </button>
  </div>
)}
```

The `useTransition` for bulkMoveJobs can use the existing `startTransition` already in the component.
